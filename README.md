# Angular Access Control

## Configuração:
- Instalar o Node.js
- Instalar o modulo "http-server". -> npm install -g http-server
- Entrar na pasta do projeto "app" pelo terminal. 
- executar o comando "http-server" 
- Abrir link no navegador "http://localhost:8080/".
